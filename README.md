# 중앙대학교 학생 취업 준비 과정의 정서 분석과 맞춤형 지원 제도 설계 제안

> 에브리타임 '취업·진로' 게시판 1년치 크롤링과 자체 학내 설문조사를 결합하여 중앙대학교 학생들의 취업 준비 정서를 텍스트 마이닝으로 분석한 프로젝트입니다. KcELECTRA 기반 감정 분류와 LDA 토픽 모델링을 거쳐 도출한 **4개 정서 페르소나**(현실 불안형 / 경쟁 민감형 / 피로 공감형 / 준비 좌절형)에 맞춰, 학내 '레인보우 시스템' 접근성 개선과 AI 챗봇 기반 통합 지원 앱('구해줘! 잡스')까지 제도 단위 제안을 함께 도출했습니다.

## 2. 분석 요약 (Key Findings)

- **학내 스트레스 정량화** — 자체 설문에서 중앙대 학우의 **약 85.4%** 가 중간(3점) 이상 수준의 취업 준비 스트레스를 경험한다고 응답.
- **공채 시즌 정점 확인** — 월별 부정(negative) 게시글 빈도 추이가 **공채 시즌과 일치**, 정서 부담이 외부 일정에 강하게 동조됨을 확인.
- **4개 정서 페르소나 도출** — 부정 감정 게시글에 LDA(4 토픽) + 감정 강도(0~1)를 결합해 클러스터링.
  - **현실 불안형** — 취업 이후 조건에 대한 불확실성
  - **경쟁 민감형** — 지원 과정에서 타인과의 비교로 인한 조급함·열등감
  - **피로 공감형** — 명확한 주제는 없지만 피로·무기력·공감형 하소연이 다수
  - **준비 좌절형** — 스펙 부족·반복 탈락 등 준비 과정의 심리적 좌절
- **제도 단위 제안** — 페르소나별 맞춤 프로그램 + '레인보우 시스템' 접근성 개선안 + AI 챗봇 기반 통합 지원 앱('구해줘! 잡스') 구상까지 분석 결과를 정책·운영 액션으로 연결.

## 3. 배경 및 목적

최근 한국 사회는 고용 불안과 경기 침체로 청년 실업률이 높은 수준을 유지하고 있으며, 많은 대학생이 학업과 취업 준비를 병행하며 큰 스트레스를 겪고 있습니다. 중앙대학교 학우를 대상으로 한 자체 설문조사에서도 약 **85.4%** 가 중간 수준(3점) 이상의 스트레스를 경험한다고 응답했습니다.

본 연구는 중앙대학교 학생의 취업 준비 과정에서 나타나는 정서적 반응을 텍스트 마이닝으로 파악하고, 그 결과를 바탕으로 다음 두 가지 결과물을 함께 도출하는 것을 목표로 합니다.

1. **정서 기반 맞춤형 취업 지원 방안** — 페르소나별 적합 프로그램 매칭
2. **'레인보우 시스템' 접근성 개선책** — 학내 기존 취업 지원 인프라의 UX 개선과 통합 앱 구상

## 5. 분석 흐름

### 5.1 데이터 수집 (크롤링)

Selenium 기반 크롤러로 에브리타임 '취업·진로' 게시판 1년치를 수집했습니다 (`crawlers/에타_크롤링.py`).

### 5.2 데이터 전처리
1. 중복 글 제거
2. 텍스트 정제 (특수문자·이모지 처리)
3. **형태소 분석(Mecab-ko)** 으로 사용자 정의 복합어 병합, 주요 품사 추출, 사용자·플랫폼 고유어·불용어 제거

### 5.3 감정 분석

`beomi/KcELECTRA-base-v2022` 모델 기반 감정 분류 (긍정 · 중립 · 부정).

- 텍스트와 감정의 수동 라벨링 → 모델 파인튜닝 → 전체 데이터 예측.
- 추가 비교: KoBERT / `beomi/kcbert-large` 분류기 학습·추론 (`notebooks/03_감정_분류.ipynb`).

### 5.4 토픽 모델링 (LDA)

감정 분류 결과 *부정(negative)* 으로 분류된 게시글에 한정하여 **LDA(4 토픽)** 를 적용. 중앙대생이 취업 준비 과정에서 어떤 주제에 부정적 감정을 표현하는지 추출합니다 (`notebooks/02_부정_토픽_모델링.ipynb`, pyLDAvis 시각화).

### 5.5 클러스터링 — 4개 정서 페르소나

토픽 분포(LDA 4차원 확률 벡터)와 부정 강도(Transformers 분류기 negative 확률 0~1)를 결합한 5차원 벡터를 클러스터링하여 4 페르소나를 정의합니다.

| 페르소나 | 특징 |
|---|---|
| 현실 불안형 | 취업 이후 조건에 대한 불확실성 |
| 경쟁 민감형 | 지원 과정에서 타인과의 비교로 인한 조급함·열등감 |
| 피로 공감형 | 명확한 주제는 없지만 피로·무기력·공감형 하소연이 다수 |
| 준비 좌절형 | 스펙 부족·반복 탈락 등으로 인한 준비 과정의 심리적 좌절 |

### 5.6 가설 검정 6건

| 가설 | 비교 단위 |
|---|---|
| 요일별 부정 감정 비율 차이 | 게시 요일 |
| 지역별 부정 감정 비율 차이 | 작성자 추정 지역 |
| 시간대별 부정 감정 비율 차이 | 게시 시각 |
| 부정 글과 긍정 글의 단어 사용 차이 | 감정 라벨 |
| 본문과 댓글 사이의 단어 사용 차이 | 텍스트 종류 |
| 추천 받은 글과 일반 글의 단어 사용 차이 | 추천 여부 |

## 6. 주요 결과

### 6.1 시간 축 — 공채 시즌 정점

월별 부정(negative) 게시글 빈도 추이가 **공채 시즌(9~11월)** 에서 두드러진 정점을 형성. 외부 채용 일정이 학생 정서 부담의 가장 강한 외부 요인 중 하나임을 정량적으로 확인했습니다.

### 6.2 정서 페르소나 4종

부정 게시글의 토픽 분포 + 부정 강도를 PCA로 시각화하면 4개 군집이 비교적 명확하게 분리됩니다 — 클러스터별 정서적 특성이 명확해 후속 프로그램 매칭의 근거로 활용됩니다.

### 6.3 자체 학내 설문조사 결과

약 **85.4%** 가 5점 척도 기준 3점 이상의 취업 준비 스트레스를 경험한다고 응답. 텍스트 마이닝 결과의 정당성을 학내 표본으로 한 번 더 검증했습니다.

## 7. 제도 설계 제안

### 7.1 페르소나별 맞춤 프로그램

| 페르소나 | 추천 프로그램 | 의도 |
|---|---|---|
| 현실 불안형 | 커리어 시뮬레이션 프로그램 | 공기업·대기업 등 경로별 커리어·삶 비교로 불확실성 완화 |
| 경쟁 민감형 | 1:1 자소서/면접 집중 케어 프로그램 | 타인 비교가 아닌 개인 능력 강화에 집중 |
| 피로 공감형 | AI 챗봇 · 학생 게시판 활성화 | 감정 토로·해소 채널 제공 |
| 준비 좌절형 | 교내외 인턴십 · 취업캠프 + 심리 케어 프로그램 | 실역량 보강과 심리 케어 병행 |

### 7.2 '레인보우 시스템' 접근성 개선

학내 취업 지원 시스템 '레인보우'의 UX 한계를 개선하기 위한 제안.

- 취업 프로그램 위주의 모바일 앱 운영
- 업종/직업별 광장, 기업별 광장으로 멘토링 접근성 향상
- 학생 개개인에게 맞춤형 취업 프로그램 로드맵 제공

### 7.3 AI 기반 통합 지원 시스템 — '구해줘! 잡스 앱'

분석 결과를 운영 형태로 통합한 가상 시스템 구상.

- 카카오톡을 통한 시기별 프로그램 알림
- 감정 케어 기능을 통한 정서적 지원까지 연계
- 페르소나 진단 → 멘토·프로그램·감정 케어 자동 연결

## 11. 한계 및 향후 과제

- **단일 학교 표본** — 중앙대 에브리타임 1년치만 사용하여 다른 대학·학년 정서 패턴으로 일반화하기 어렵습니다. 다대학 비교 분석이 필요합니다.
- **자체 설문 표본 한계** — 학우 설문은 학내 모집 기반이므로 응답 편향이 있을 수 있습니다.
- **익명 텍스트의 식별자 부재** — 같은 작성자의 시계열 정서 변화를 추적할 수 없어 페르소나는 *게시글 단위 정서 프로필* 입니다.
- **모델 외부 검증 부재** — 감정 분류기는 KcELECTRA 사전학습 + 학내 데이터 라벨링 파인튜닝으로 학습됐으나, 별도 외부 라벨 데이터셋에서의 일반화 성능은 추가 검증이 필요합니다.
- **제도 제안의 검증** — 제안한 프로그램·앱 구상의 효과는 실제 운영 후 A/B 평가가 필요합니다.
